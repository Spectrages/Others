Эта программа предназначена для создания цветовой гистограммы из выбранного изображения с использованием трех каналов RGB. Если изображение находится на указанном пути, введите его полное имя (cat.jpg), в противном случае, введите в путь самостоятельно. 

This program is designed to create a color histogram from a selected image using three RGB channels. If the image is on the specified path, enter its full name (cat.jpg), otherwise, enter in the path yourself.
